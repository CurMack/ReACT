ReACT : Reactive Bimanual Robot Control
via Action Chunking Transformer

Abstract

Generating effective and context-aware reactions for robot manipulation tasks remains challenging, particularly in bimanual operations requiring precise coordination. This project presents a low-cost teleoperation system leveraging ALOHA (A Low-cost Open-source Hardware Architecture), which allows an operator to teleoperate one robot arm while training another to autonomously generate complementary reactions. Our approach employs Action Chunking with Trans formers (ACT), a novel learning framework designed for high- frequency, fine-grained manipulation tasks. ACT encodes hu man demonstration sequences into temporally cohesive chunks, mitigating compounding errors common in imitation learning, while ensuring smooth and precise motions. Using our system, we successfully train one robot arm to react autonomously in real-world tasks, such as collaborative bimanual assembly and object handling, with high accuracy. This work highlights the potential of combining low-cost hardware and advanced learn ing algorithms to democratize high-precision robotic systems.

Overview

Given the target scene RGB-D image,AffordDP retrieves a similar object in the affordance memory and transfers its static and dynamic affordance to the target object. Conditioned on 3D affordance, point cloud observation, and robot proprioception, AffordDP utilizes the Diffusion Policy and adaptive affordance guidance for precise control.
